# Deque Demo Qt

Интерактивное приложение на Qt для демонстрации работы с контейнером std::deque и алгоритмами STL.

## Описание

Приложение для демонстрации работы с контейнером std::deque и алгоритмами STL. Реализован визуальный интерфейс для выполнения операций с контейнером и применения различных алгоритмов.

## Функциональность

- Визуализация работы с std::deque
- Демонстрация STL алгоритмов:
  - Merge Sort (сортировка слиянием)
  - Find (поиск элемента)
  - Count (подсчет элементов)
  - Min/Max Element (поиск минимального/максимального)
  - Unique (удаление дубликатов)
  - Reverse (обращение последовательности)
  - Shuffle (перемешивание)
  - Upper/Lower Bound (поиск границ)
- Работа с итераторами
- Интерактивный интерфейс

## Технологии

- **C++20** - основной язык программирования
- **Qt Framework** - для создания GUI
- **std::deque** - демонстрируемый контейнер
- **STL алгоритмы** - различные алгоритмы стандартной библиотеки
- **Итераторы** - работа с итераторами контейнера

## Требования

- Qt 5.12+ или Qt 6.x
- Компилятор с поддержкой C++20
- qmake или CMake

## Сборка

### Использование qmake

```bash
qmake submission.pro
make
```

### Использование CMake

```bash
mkdir build
cd build
cmake ..
make
```

## Использование

### Запуск приложения

```bash
./deque_demo
```

### Доступные операции

- **push_front** - добавление элемента в начало
- **push_back** - добавление элемента в конец
- **pop_front** - удаление элемента из начала
- **pop_back** - удаление элемента с конца
- **insert** - вставка элемента
- **erase** - удаление элемента
- **clear** - очистка контейнера

### Доступные алгоритмы

- **Merge Sort** - сортировка слиянием
- **Find** - поиск элемента
- **Count** - подсчет элементов
- **Min Element** - поиск минимального элемента
- **Max Element** - поиск максимального элемента
- **Unique** - удаление дубликатов
- **Reverse** - обращение последовательности
- **Shuffle** - перемешивание элементов
- **Upper Bound** - поиск верхней границы
- **Lower Bound** - поиск нижней границы

## Особенности реализации

- **Визуализация**: наглядное отображение операций с контейнером
- **Интерактивность**: удобный интерфейс для экспериментов
- **Образовательная ценность**: демонстрация работы STL алгоритмов
